# KasRW — Sistem Pencatatan Kas RW Digital

Aplikasi web microservice untuk pencatatan kas RT/RW yang transparan dan akuntabel.
Dideploy di AWS EC2 sebagai bagian dari Tugas Besar II Cloud Computing.

## Tech Stack
- **Backend**: Node.js v20 + Express.js + Prisma ORM
- **Frontend**: React 18 + Vite + Tailwind CSS
- **Database**: PostgreSQL 15
- **Deployment**: AWS EC2 Ubuntu 24.04 LTS + Nginx + PM2

## Struktur Project
|
|---backend/ --> Express.js API server (port3001)
|
|---frontend/ --> React SPA (build -> static)
